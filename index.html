Longer description of the game

Navigate through 9 stages, each featuring different weather conditions and enemies, by selecting units, creating player skills through a poker game, and strategically placing units. The units are characters from board games, while the enemies are characters from digital games. Protect the Board Game Kingdom from the digital game invasion by placing your units to prevent enemies from escaping the tiles.

You can enhance each unit. As you progress through the stages, utilize the upgrade function to strengthen your units. Your strategy is crucial to defeating the enemies.

Gameplay Description: How to Play

You are the hero destined to save the Board Game Kingdom. First, select the units you will use for defense. Each stage has different units available and varying weather conditions, so strategic unit selection is crucial. Refer to the window that shows the attributes of each unit.

After choosing your units, you can select player skills using poker cards. The higher the poker hand, the more powerful the player skills you will obtain. If you are unfamiliar with poker rules, consult the poker book at the top.

Once the game starts, you can place your units. Each unit can be used according to its attack power and cooldown time. If an enemy escapes the tile, the game is over. If you defeat all the enemies, you win. The currency earned in-game can be used for the next stage.
